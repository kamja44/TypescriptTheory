# 2.1

- 데이터와 변수의 타입을 명시적으로 정의 가능
  - 변수가 비어있을때만 사용(타입을 추론한 코드와 결과가 동일하다.)
    - 즉, 타입스크립트가 타입을 추론하지 못할 때 사용한다.

```js
ㅣet b : boolean = true
```

- JS처럼 변수만 선언해도됨
  - 타입스크립트가 타입을 추론함

```js
let a = "hello"; // TS가 a의 타입을 string으로 추론한다.
```
